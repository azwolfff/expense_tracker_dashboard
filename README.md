

A Java Swing-based desktop application to manage daily expenses with a clean and simple dashboard interface.

## Features

* Add and delete expenses
* Category selection
* Date input
* Monthly expense calculation
* Search and filter system
* Dashboard summary (Total, Monthly, Entries)

## Technologies Used

* Java
* Swing (GUI)

## How to Run

1. Open project in IntelliJ IDEA
2. Run the main Java file
